# User Flow & Use Case Map: Media Narrative Tracker This document maps the screens designed for the Media Narrative Tracker to the core user flows and use cases defined in the Product Requirements Document. --- ## Flow 1: Exploration & Creation **Goal:** Define a new area of intelligence from scratch. * **Step 1: Search Library Dashboard (HomeScreen)** * **Use Case:** The analyst starts their day here to see overall volume across all active searches and navigate to specific collections. * **Action:** Clicks "New Search". * **Step 2: Search Create / Edit Form** * **Use Case:** Building the "Base Search". Defining high-level keywords (e.g., "Lithium Supply") and date ranges. * **Action:** Deploys the search to the engine. --- ## Flow 2: Enrichment & Composable Search **Goal:** Refine a base search into a specific, high-signal derivative without losing the original context. * **Step 3: Search Detail & Filter Editor** * **Use Case:** Investigating the results of a base search. Layering on specific filters to narrow down the narrative. * **Action:** Clicks "Fork Search" to create a versioned instance. * **Step 4: Fork Search Modal** * **Use Case:** Explicitly inheriting the parent's logic while creating a new, isolated branch (e.g., "Lithium Supply - Negative Sentiment Only"). * **Action:** Saves the new derivative. * **Step 5: Search Lineage Explorer** * **Use Case:** Visualizing the Neo4j graph relationships. Understanding the "provenance" of an insight by seeing which parent search a derivative came from. --- ## Flow 3: Reusability & Scaling **Goal:** Apply consistent logic across multiple searches efficiently. * **Step 6: Filter Preset Library & Modal** * **Use Case:** Defining a "Tier 1 Sources" filter once and applying it to 12 different searches. * **Impact:** Ensures consistency in methodology across the entire analyst team. * **Step 7: Collection Management Modal** * **Use Case:** Grouping related searches (Base + Derivatives) into a single "Collection" for stakeholder reporting or project organization. --- ## Flow 4: Insight & Monitoring **Goal:** Moving from "matching articles" to "narrative signals." * **Step 8: Narrative Trends & Insights** * **Use Case:** The final output. Analyzing coverage volume spikes, sentiment shifts over time, and co-occurring topics to identify emerging risks or opportunities. * **Outcome:** Actionable intelligence derived from the entire structured search workflow.
